HTML TAGS

No. Tag Name Tag Description Code Example
1 html Defines the root of an HTML document <html></html>
2 head Defines information about the document <head></head>
3 title Defines a title for the document <title></title>
4 body Defines the document's body <body></body>
5 h1 to h6 Defines HTML headings <h1></h1>,<h2></h2> ....
6 p Defines a paragraph <p></p>
7 a Defines a hyperlink <a href=""></a>
8 img Defines an image <img src="">
9 ol Defines an ordered list <ol></ol>
10 ul Defines an unordered list <ul></ul>
11 li Defines a list item <li></li>
12 div Defines a section in a document <div></div>
13 table Defines a table <table></table>
14 tr Defines a row in a table <tr></tr>
15 td Defines a cell in a table <td></td>
16 form Defines an HTML form for user input <form></form>
17 input Defines an input control <input type="" value="">
18 link Defines the relationship between a document and an external resource (most used to link to style sheets) <link rel="stylesheet" type="text/css" href="">
19 iframe Defines an inline frame <iframe src=""></iframe>
20 textarea Defines a multiline input control (text area) <textarea></textarea>
21 select Defines a drop-down list <select></select>
22 b Defines bold text <b></b>
23 style Defines style information for a document <style type="text/css"></style>
24 strong Defines important text <strong></strong>
25 button Defines a clickable button <button></button>
